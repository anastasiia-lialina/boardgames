# Boardgames

### Описание
Каталог настольных игр с отзывами, рейтингом и организацией встреч

### Технологии
- PHP 8.2+ Yii2 Framework
- PostgreSQL 14
- Redis 7
- Docker + Docker Compose
- Nginx + PHP-FPM

### Архитектура
Проект следует паттерну MVC с дополнительным сервисным слоем для бизнес-логики:

- **Controllers** - обработка HTTP запросов и управление потоком
- **Services** - бизнес-логика и работа с данными
- **Models** - ActiveRecord модели и правила валидации
- **Forms** - формы для валидации пользовательского ввода

### Основные возможности
- Каталог игр с фильтрацией и поиском
- Система отзывов с модерацией
- Организация игровых сессий
- Подписки на игры
- Управление пользователями и ролями

### Установка и запуск

#### Через Docker (рекомендуется)
```bash
# Клонирование репозитория
git clone <repository-url>
cd boardgames

# Запуск контейнеров
make up

# Установка зависимостей
make composer-install

# Выполнение миграций
make migrate
```

#### Вручную
```bash
# Установка зависимостей
composer install

# Настройка окружения
cp .env.example .env
# Отредактируйте .env файл с вашими настройками

# Миграции базы данных
php yii migrate

# Запуск сервера разработки
php yii serve
```

### Доступные команды
```bash
make up              # Запуск Docker контейнеров
make down            # Остановка контейнеров
make logs            # Просмотр логов
make composer-install # Установка Composer зависимостей
make migrate         # Выполнение миграций
make test            # Запуск тестов
make fix             # Форматирование кода
```

### Структура проекта
```
/
├── controllers/          # Контроллеры
├── models/             # Модели данных
│   ├── game/          # Модели игр
│   ├── user/          # Модели пользователей
│   └── forms/         # Формы валидации
├── services/           # Сервисный слой
├── views/              # Шаблоны представлений
├── messages/           # Файлы локализации
├── migrations/         # Миграции БД
├── config/            # Конфигурация приложения
└── docker/            # Docker конфигурации
```

### Тестирование
```bash
# Запуск всех тестов
make test

# Запуск конкретного теста
vendor/bin/phpunit tests/unit/SomeTest.php
```

### Код стиль
Проект использует PHP CS Fixer для поддержания единого стиля кода:

```bash
# Проверка стиля
make check-style

# Исправление стиля
make fix
```
